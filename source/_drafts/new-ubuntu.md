## 安装 zsh
```
apt install zsh
```

设置默认 shell 
```
chsh -s /bin/zsh
```

## 安装 on-my-zsh (github)

[https://github.com/robbyrussell/oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)
```
sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
```
